# Challenge #49

What does this Linux function do?

```assembly
main:
    pushq   %rbp
    movq    %rsp, %rbp
    movl    $2, %edi
    call    sleep
    popq    %rbp
    ret
```

---

This function calls `sleep(2)`, which sleeps for 2 seconds, then returns.
